# DARTBOT
## Telegram DART streaming service 
## version: 4.0.0
## services: 
 1. [대량보유 상황보고]: 주식등의 대량보유상황보고서 내에 대량보유 상황보고 정보를 제공합니다.
 2. [임원ㆍ주요주주 소유보고]: 임원ㆍ주요주주특정증권등 소유상황보고서 내에 임원ㆍ주요주주 소유보고 정보를 제공합니다.
